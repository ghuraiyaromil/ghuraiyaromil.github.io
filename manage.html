// COMPLETE UPLOAD GUIDE

  
Step-by-step process for uploading all files to your GitHub repo so they appear on your live website.


  

    

      
1

      

        
Go to your GitHub repo

        
Open github.com/ghuraiyaromil/ghuraiyaromil.github.io in your browser and make sure you're logged in.

      

    

    

      
2

      

        
Click "Add file → Upload files"

        
This button is on the main repo page, above the file list on the right side.

      

    

    

      
3

      

        
Type the folder path before uploading

        
At the top of the upload page you'll see a file path box. Click inside it and type the folder name with a slash before you drop your files. GitHub will create the folder automatically.

        
For gallery photos → type: images/

        
For certificates → type: certificates/

        
For reports/PDFs → type: reports/

      

    

    

      
4

      

        
Drag and drop your files

        
Drop all files for that folder at once. You can upload multiple files in one go. Make sure the filenames match exactly what you typed in the code generator above (case-sensitive).

      

    

    

      
5

      

        
Commit the upload

        
Scroll down, leave the commit message as-is or type something like "Add gallery images", select "Commit directly to main branch", then click the green Commit button.

      

    

    

      
6

      

        
Update index.html with the generated code

        
Go back to your repo, click index.html, then click the pencil icon (Edit). Paste the code you generated from this tool into the right section. Commit when done.

      

    

    

      
7

      

        
Wait 1–2 minutes, then check your live site

        
Open https://ghuraiyaromil.github.io — GitHub Pages takes about 1–2 minutes to rebuild after every commit.
